## Upgrading

Follow these steps to update HelloToms to the latest version.

<Steps>
  <Step title="Backup Your Site" icon="save">
    Commit your current changes to Git.

    ```bash
    git add .
    git commit -m "Backup before upgrade"
    ```
  </Step>
  <Step title="Update Dependencies" icon="package">
    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn,pnpm">
      ```bash
      npm install next@latest tailwindcss@latest framer-motion@latest
      ```
      ```bash
      yarn add next@latest tailwindcss@latest framer-motion@latest
      ```
      ```bash
      pnpm add next@latest tailwindcss@latest framer-motion@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Run Migrations" icon="play">
    Execute the upgrade script and restart dev server.

    ```bash
    npm run migrate
    npm run dev
    ```
  </Step>
</Steps>
