## Overview

HelloToms is your ready-to-use personal portfolio starter kit. It helps you showcase your software engineering expertise in web development, mobile applications, and cloud solutions. Built with Next.js, Tailwind CSS, and MDX for documentation, HelloToms delivers a professional, responsive site that highlights your projects, skills, and experience. Customize it quickly to reflect your unique brand and launch in minutes.

<Callout kind="info">
  HelloToms uses MDX for rich, interactive documentation pages like this one.
</Callout>

## Brand Guidelines

Follow these guidelines to maintain consistency.

<ExpandableGroup>
  <Expandable title="Primary Color" default-open="true">
    Use the brand color `#c53544` for accents, buttons, and links. Pair it with neutral grays for readability.
  </Expandable>
  <Expandable title="Typography">
    Headings use Inter font (bold weights). Body text is Inter Regular at 16px.
  </Expandable>
  <Expandable title="Logo Usage">
    Place the HelloToms logo in the header. Do not stretch or recolor it.
  </Expandable>
</ExpandableGroup>

<Callout kind="tip">
  Test your customizations in dark and light modes to ensure accessibility.
</Callout>

## Get Started in Minutes

Follow these steps to launch your portfolio.

<Steps>
  <Step title="Clone and Install" icon="download">
    Get the starter kit and dependencies.

    <CodeGroup tabs="npm,yarn">
    ```bash
    git clone https://github.com/yourusername/hellotoms.git
    cd hellotoms
    npm install
    ```
    ```bash
    git clone https://github.com/yourusername/hellotoms.git
    cd hellotoms
    yarn install
    ```
    </CodeGroup>
  </Step>
  <Step title="Customize Content" icon="edit-3">
    Update `data/projects.json` with your real projects.

    ```json
    {
      "projects": [
        {
          "title": "E-Commerce App",
          "tech": ["React", "Node.js", "AWS"],
          "demo": "https://demo.example.com"
        }
      ]
    }
    ```
  </Step>
  <Step title="Deploy" icon="rocket">
    Deploy to Vercel or Netlify.

    <Tabs>
      <Tab title="Vercel" icon="cloud">
        Connect your GitHub repo and deploy with one click.
      </Tab>
      <Tab title="Netlify" icon="globe">
        Drag your build folder or link GitHub for CI/CD.
      </Tab>
    </Tabs>
  </Step>
</Steps>
